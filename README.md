# Annual Sales Performance Dashboard using Power BI

This Power BI project analyzes Customer and Order data to identify revenue trends, customer behavior, and regional performance.

The dashboard incorporates a range of visualizations, including stacked column charts, stacked bar charts, donut charts, and data cards, to present insights effectively. It also includes key modifications such as filters, visual effects, and DAX calculations for deriving meaningful metrics, along with interactive elements like slicers and a map chart to enable dynamic analysis.

## Insights Derived
- Monthly profit trends and fluctuations
-	Top-performing states, product categories & sub-categories
-	Quarterly and state-wise data trends to identify performance patterns
-	High-value customers contributing to revenue

## Key Insights
- November was the highest profit-making month while the company suffered losses in May, July, September, and December.
- States generated revenue: Maharashtra, Madhya Pradesh, and Uttar Pradesh.
- Top spending customers belong from New Delhi and Mumbai city.
- Cash on Delivery is the highest chosen mode of payment.
